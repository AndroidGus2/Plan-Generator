This is a plan for organizing a party.

This is what is going to be acomplished in this plan: To host a birthday party

These are the issues that could occur during the plan and how they will be dealt with:

  • The original cake maker cannot make the cake. It can be solved by having multiple backup chefs who can make the cake in case the original chef cannot make the cake

  • No place for the party to be hosted which can be dealt with by picking a location that can have reservations so that you can gurantee that you can use it or pick a pick a location that can adhere to covid restrictions.

  • Guests not being able to come to the party. The best solution is to give guests their invites as early as possible so they canmake sure that they can attend.
